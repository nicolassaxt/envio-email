# MICROSERVICE DE ENVIO DE EMAIL

Esse é um micro serviço criado para o envio de e-mail feito em Java e utilizando o RabbitMQ. 



## Requirements

- Java 8

- RabbitMQ

  

## Spring Boot

- https://start.spring.io/
- Java 8
- Gradle Project
- Spring Web
- Spring Data JPA
- Amqp

- Mysql Connector
